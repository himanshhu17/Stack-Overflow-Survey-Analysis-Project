# 📊 Stack Overflow Survey Analysis Project

## 🚀 Overview
This project analyzes the Stack Overflow Developer Survey dataset to uncover insights about developer trends, technologies, salaries, and preferences across the global tech community.

The goal is to perform data cleaning, exploratory data analysis (EDA), and visualization to extract meaningful patterns that help understand the software industry.

---

## 🎯 Objectives
- Analyze developer demographics (age, country, education)
- Identify most popular programming languages and tools
- Study salary trends across regions and experience levels
- Understand developer preferences and work environments
- Generate insights using data visualization

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
- Source: Stack Overflow Developer Survey (public dataset)
- Format: CSV
- Includes:
  - Developer demographics  
  - Programming languages  
  - Tools & technologies  
  - Salary data  
  - Work preferences
